# interactive-player

## Description
A front-end interactive music audio player using AudioWorklet for javascript and typescript.

## Features

- [x] Play/Stop
- [x] Pause/Resume
- [x] Interactive Music Horizontal Transition
- [ ] Interactive Music Vertical Transition
- [x] FadeIn/FadeOut
